# SkillOpus

Welcome to SkillOpus (formerly EduLearn).

This repository contains:
- **Frontend**: The React-based user interface.
- **Backend-Mongo**: The Node.js/Express backend with MongoDB.

## Getting Started

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

### Backend
```bash
cd Backend-Mongo
npm install
npm start
```
